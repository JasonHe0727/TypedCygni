# Types

## Basic Types

Int, Float, Long, Double, Boolean, Char, String

```
Int + Int -> Int
Float + Float -> Float
Long + Long -> Long
Double + Double -> Double

Int == Int -> Boolean
Float == Float -> Boolean
Long == Long -> Boolean
Double == Double -> Boolean
Boolean == Boolean -> Boolean
Char == Char -> Boolean
String == String -> Boolean
```

## Built-in Functions

```
Length("abc") -> 3
Length(Array[Int](1, 2, 3)) -> 3
As[Int](3.2) -> 3
Is[Int](3.2) -> false
```
